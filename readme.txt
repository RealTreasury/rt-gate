A WordPress plugin for admin-managed gated content with form-to-asset mapping, tokenized access, and event tracking.

== Description ==

Real Treasury Gate enables site administrators to:

* Create and manage lead-capture forms.
* Map forms to one or more gated assets (downloads, videos, and links).
* Generate token-based access URLs for iframe-hosted gated pages.
* Track engagement events such as downloads and video milestones.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/real-treasury-gate` directory, or install via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure forms, assets, and mappings from the Real Treasury Gate admin menu.

== Frequently Asked Questions ==

= Does this plugin support multiple forms and assets? =

Yes. You can define multiple forms and map each form to one or many assets.

= How is access controlled? =

Access is controlled with expiring, token-based URLs associated with specific assets.
